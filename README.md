# Sentiment Analysis of YouTube Comments for Free Nutritious Meal Program Using IndoBERT and AutoML

## Overview

This project focuses on sentiment analysis of Indonesian YouTube comments related to the **Free Nutritious Meal Program (Makan Bergizi Gratis / MBG)**. The objective of this research is to classify public opinions into three sentiment categories: **positive, neutral, and negative**.

The analysis combines **IndoBERT** as a transformer-based language model for Indonesian text representation and **AutoML (TPOT)** to automatically search and optimize machine learning pipelines for sentiment classification.

The developed model aims to understand public responses and provide insights regarding public perception toward the Free Nutritious Meal Program.
